> Chrome extensions are updated automatically as a new version is available in Chrome Store.

Chrome takes some time to update your installed extensions. It could range between 1-2 hours to 1-2 days.

**To manually update the extension** you can follow these steps:

1. Go to URL: `chrome://extensions/`

2. Turn on developer mode

   <Image src="https://imagedelivery.net/IEMzXmjRvW0g933AN5ejrA/wwwnotionso-image-prod-files-secures3us-west-2amazonawscom-8e04dd23-74cd-4c44-8158-4238eac359d1-ae2a9849-a019-4d15-a3d7-a787d93ebc43-untitledpng/public" align="left" />

3. Search “Requestly” in the Top Search Bar and Click on Requestly Card

4. Click on the update button

   <Image src="https://imagedelivery.net/IEMzXmjRvW0g933AN5ejrA/wwwnotionso-image-prod-files-secures3us-west-2amazonawscom-8e04dd23-74cd-4c44-8158-4238eac359d1-aa4e6955-cf53-4d5f-a41c-0bf7d1dabe8a-untitledpng/public" align="left" />

This should update Requestly to latest version.
