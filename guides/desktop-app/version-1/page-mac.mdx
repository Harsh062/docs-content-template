## Installation

Follow these steps to install Requestly Desktop App on MacOS.

1. Download Requestly Desktop App for Mac from [here](https://requestly.io/desktop). This should download the latest dmg of the Desktop App.

2) Double click on the dmg downloaded. You should see a window popup.

Drag the `Requestly Icon` to `Applications`. This should install Requestly to your Mac.
