Once you open a session book from extension popup, you may replay the video and review all the details like Console logs, Network logs and Environment details.

### Setting start time

You may update the start time of the video to skip the irrelevant portion.

<Image src="https://imagedelivery.net/IEMzXmjRvW0g933AN5ejrA/wwwnotionso-image-s3-us-west-2amazonawscom-securenotion-staticcom-06720150-5c80-4443-8f71-2b71ab4e31e5-untitledwebp/public" align="left" />

For any user who is viewing the recording once it is shared, video will start playing after this time offset.

### Adding description

You may describe the bug or add important information regarding this recording in the description box.

<Image src="https://imagedelivery.net/IEMzXmjRvW0g933AN5ejrA/wwwnotionso-image-s3-us-west-2amazonawscom-securenotion-staticcom-64fa2413-03fe-47e1-b04d-409bc4e6c783-untitledwebp/public" align="left" />

### Save

If you are satisfied with the recording, you may click **Save Recording** button on the top. You can either save online or save locally in a file format.

<Image src="https://imagedelivery.net/IEMzXmjRvW0g933AN5ejrA/wwwnotionso-image-s3-us-west-2amazonawscom-securenotion-staticcom-97de5bae-b237-4462-9e36-52579988c4aa-untitledpng/public" align="left" />
