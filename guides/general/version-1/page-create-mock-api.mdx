2) Click on **Create New** OR you can **Upload** a JSON file too.

   <Image src="https://imagedelivery.net/IEMzXmjRvW0g933AN5ejrA/wwwnotionso-image-prod-files-secures3us-west-2amazonawscom-8e04dd23-74cd-4c44-8158-4238eac359d1-8f5eadaa-5078-4558-9048-30a20ec7edbf-ss-2_35png/public" align="left" />

3. Enter the details of the Mock API and then click on **Create**.

   <Image src="https://imagedelivery.net/IEMzXmjRvW0g933AN5ejrA/wwwnotionso-image-prod-files-secures3us-west-2amazonawscom-8e04dd23-74cd-4c44-8158-4238eac359d1-34855f8d-3d10-4043-999b-8085703c2894-ss-2_36png/public" align="left" />

Your Mock API is ready now. Copy the URL to see it in action!
