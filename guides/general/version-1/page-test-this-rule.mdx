You can test your rule using this feature. It enables you to check if the rule is configured correctly and is working as expected on a web page.

<Image src="https://i.ytimg.com/vi/heej7VXk33k/hqdefault.jpg" align="left" />

### Steps to test this rule

1. **Create a rule: **Before testing a rule, ensure that the rule you have created has been saved and enabled.

2) **Enter the test URL**: After saving the rule, a `test this rule` panel will show up automatically. Enter the page URL where you want to test the rule you have created and click on **Test rule **button.

3. **Record Session**: You can configure if you want to record a session when testing.

   1. <Image src="https://imagedelivery.net/IEMzXmjRvW0g933AN5ejrA/wwwnotionso-image-prod-files-secures3us-west-2amazonawscom-8e04dd23-74cd-4c44-8158-4238eac359d1-37eb09f1-8e0a-4e8d-b6ee-24d710cd210b-ss-2_10png/public" align="left" />

4) **Observe rule application: **After starting the test, you will be redirected to a new tab with the URL you had entered. A widget will be shown which will notify if the the rule has been applied yet or not. The widget updates as soon as the rule is applied.

5. **View test results: **When youâ€™re done with testing you can click on **view results **button which will generate a test report in the rule editor.

   1. <Image src="https://imagedelivery.net/IEMzXmjRvW0g933AN5ejrA/wwwnotionso-image-prod-files-secures3us-west-2amazonawscom-8e04dd23-74cd-4c44-8158-4238eac359d1-d0d5f33b-ae6a-4f7c-a8b2-b35cf12267a0-ss-2_11png/public" align="left" />

6) **View test session**: You can get more details about the test report by viewing the network and console logs of the testing session by clicking on **Watch session **button.

   1. <Image src="https://imagedelivery.net/IEMzXmjRvW0g933AN5ejrA/wwwnotionso-image-prod-files-secures3us-west-2amazonawscom-8e04dd23-74cd-4c44-8158-4238eac359d1-bf7a7a2e-7b13-4027-9b16-ad6bb989723a-ss-2_13png/public" align="left" />
