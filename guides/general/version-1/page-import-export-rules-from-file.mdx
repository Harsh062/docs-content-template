You can import rules from JSON file or export them as JSON file hassle free.

### Steps to import

1. Go to [https://app.requestly.io/rules/my-rules](https://app.requestly.io/rules/my-rules)

2) Under the HTTP rules section, click on `Import` .

1. Upload the JSON File.

### Steps to Export

1. Select one or more rule to be exported.

2) Click on `Share` button .

3. A share modal will popup, switch to `Download` tab and click on **Download **button.

4) Your rules will be exported as a JSON file.
